# Corpus de fichiers reels

Deposer ici des fichiers de donnees reels (`.csv`, `.tsv`, `.txt`, `.xlsx`,
`.xls`), **anonymises**, representatifs des cas rencontres en pratique
(en-tetes sur plusieurs lignes, encodages varies, codes de manquant
heterogenes, feuilles Excel multiples, etc.).

Ce repertoire est volontairement vide au depart : la campagne de tests
(`run_corpus_tests()`, dans `tests/corpus/run_corpus.R`) n'a de sens
qu'avec de vrais fichiers difficiles, jamais avec des exemples fabriques
pour passer.
